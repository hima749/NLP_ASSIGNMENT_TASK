# Sentiment Analysis using NLP (IMDB Dataset)

This project focuses on performing sentiment analysis on movie reviews using Natural Language Processing (NLP) techniques. The goal is to classify text data as positive or negative based on user reviews.

## 📌 Dataset

* IMDB Movie Reviews Dataset
* Source: Kaggle 
* Contains labeled reviews (positive / negative sentiment)

## 📌 Features Implemented

* Text preprocessing and cleaning
* Tokenization and normalization
* Stopword removal
* Feature extraction (e.g., Bag of Words / TF-IDF)
* Sentiment classification (positive vs negative)
* Model training and evaluation

## 🧠 Concepts Used

* Natural Language Processing (NLP)
* Text preprocessing
* Feature engineering
* Machine Learning for classification
* Model evaluation

## ⚙️ Workflow

1. Load dataset
2. Clean and preprocess text
3. Convert text into numerical features
4. Train classification model
5. Evaluate model performance

## 📂 Files

* Jupyter Notebook → full implementation of preprocessing and model

## 🎯 Objective

To build a machine learning model capable of understanding and classifying human sentiment from textual data.

## 🚀 Outcome

Successfully developed a sentiment analysis system that can classify reviews as positive or negative, demonstrating practical NLP and machine learning skills.

## 💡 Applications

* Review analysis
* Social media monitoring
* Customer feedback analysis
* Recommendation systems
